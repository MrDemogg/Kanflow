## Kanflow
Приложение Kanflow это учебный проект, <br/> целью которого является реализация гибкой Kanban-доски для визуализации и управления задачами
<br/>
<br/>
<img align="right" width="300" height="300" alt="web_development_trello_agile_kanban_icon_265401" src="https://github.com/user-attachments/assets/cfc37fe6-4d60-4441-a766-083131099b3b" />
Суть любой Kanban-доски - разделение задач на несколько категорий, что визуально и психологически упрощает взаимодействие с ними
  
Однако Kanflow также позволяет:
* создавать произвольное количество столбцов с настраиваемыми названиями  
* добавлять, перемещать и редактировать задачи внутри столбцов  
* назначать ответственных исполнителей для каждой задачи  
* оставлять комментарии и вести обсуждение прямо на карточке  
* открывать отдельные столбцы или конкретные задачи в независимых окнах с возможностью их закрепления на рабочем столе  
  
Проект реализуется с акцентом на удобство взаимодействия, чистый интерфейс 
и возможность дальнейшего расширения функциональности.
